Vbot

基于 php 7 的 web 微信机器人

  • 支持 API

    通过 API，更方便的打造属于自己的网页版微信

  • 多种消息类型

    让你的程序更加灵活处理

  • 丰富的监听器

    能够在程序各个阶段做好各自的自定义处理

  • 稳定执行

    稳定，是 vbot 的核心基础

  • mkdir vbot
  • cd vbot
  • composer require hanson/vbot
